Online Flowchart

A tool that creates flowcharts based on text inputs. It doesn't require mouse operations. You can create if-else charts with just simple scripts. If you need to create a simple chart, this tool would be good for you.

It's easy to use. Firstly, fill in the textform with a script using the examples page as a reference. Once finished input, press the build button to check the generated chart. You can save the chart as either a web page or a pdf.
